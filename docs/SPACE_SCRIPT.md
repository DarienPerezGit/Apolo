# Apolo — BNB Chain / X Space Pitch (3-5 mins)

## 0:00 - 0:45 | The Hook
"In crypto, whenever someone explains payments, they use the coffee example. But let me propose something more interesting — almost the entire software market still lives on Web2. Why? Not because the technology isn't ready. Because nobody removed the friction to pay based on outcomes."

"This is **Apolo**. Apolo is an automated bounty and escrow infrastructure for the Agentic Economy. We let users say: 'Pay this developer when my API returns HTTP 200'. No more promises. Programmable assertions."

## 0:45 - 1:30 | The Core Problem
**"Agents can do work but can't get paid automatically based on verified outcomes."** 

"Hay millones de dólares en pagos por trabajo operativo verificable, pero no existe un sistema estándar que pague automáticamente basado en evidencia externa verificable. Apolo cubre ese gap." 

## 1:30 - 2:30 | How It Works & Demo Flow
"Apolo is SLA-verified payment infrastructure built on the BNB Chain. 
Here is how it works:
1. Client locks funds in escrow (BSC Testnet)
2. Off-chain verification checks the SLA condition (API uptime / response / evidence)
3. Trusted relayer triggers settlement on-chain
4. Funds are released or refunded automatically"

"All signing is offchain and gasless for the client. A single wallet interaction is required for execution."

## 2:30 - 3:15 | Trust Model (V1) & Evidence
"For our public V1, we maintain a Trusted Relayer model. This relayer is responsible for submitting the consensus outcome to the smart contract to execute settlement safely. We're showing a real flow on the BSC Testnet where escrow balances are transparently released to devs upon success."

## 3:15 - 4:00 | Closing
"Where is this going? With EIP-7702 coming to BNB Pascal, we have an upgrade path ready. Apolo goes beyond being just a better transaction; it is a payment primitive for Software 3.0. In a world of AI agents calling APIs and developers shipping fast, Apolo is the financial truth layer."

"Truth is programmable. Deploy. Get paid when it works. That is Apolo."
